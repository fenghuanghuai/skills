# Claude Skills Collection

This repository contains personal Claude Code skills for extending Claude's capabilities.

## Skills

### opencode-service.skill
OpenCode CLI integration guide. Use when:
- Building CI/CD pipelines with AI code review
- Creating Makefile targets for code analysis
- Setting up automated code quality checks
- Learning OpenCode CLI usage patterns

### smtp-email.skill
Send emails via SMTP protocol. Use when Claude needs to send emails programmatically for:
- Sending notifications or alerts
- Sending reports or data exports
- Automated email sending tasks
- Email-based automation workflows

## Installation

To install a skill, simply download the `.skill` file and place it in your Claude skills directory.

## Usage

Skills are automatically loaded by Claude Code based on the task context.
